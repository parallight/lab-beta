# Parallight Lab — BETA channel

Pre-release builds for invited testers. Points at the **beta** backend
(`beta.lab-agent.parallight.ai`), fully isolated from the public release.

## Install (Claude Code)

```
/plugin marketplace add parallight/lab-beta
/plugin install parallight-lab@parallight-beta
```

Then fully restart Claude Code and run `/lab-login` (your email must be on the
beta allowlist). The public, stable release lives at <https://parallight.ai/install>.
